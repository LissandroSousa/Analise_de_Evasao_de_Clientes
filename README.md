# An-lise_de_Evas-o_de_Clientes
Projeto desenvolvido como desafio de Ciência de Dados da Alura, com foco na análise de evasão de clientes em uma empresa de telecomunicações. O trabalho envolve ETL, análise exploratória, modelagem preditiva e interpretação de variáveis relevantes para apoiar estratégias de retenção.
